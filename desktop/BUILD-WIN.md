# Сборка одного EXE для Windows

## Как собрать

```bash
cd desktop
npm install
npm run build:win
```

Готовый файл будет в папке **`dist-electron/`**:
- **`RelaxDrive-Desktop.exe`** — один исполняемый файл, установка не нужна.

## Иконка на рабочем столе

1. Скопируйте `RelaxDrive-Desktop.exe` на рабочий стол.
2. Или: правый клик по файлу → «Отправить» → «Рабочий стол (создать ярлык)».

Запуск по двойному щелчку по EXE или по ярлыку.

## Требования

- Windows 10/11
- Для работы приложения нужен доступ к backend (например, указать `VITE_API_URL` в настройках или использовать по умолчанию).

## Доступ для админов (All-in-One)

Один и тот же **RelaxDrive-Desktop.exe** можно передать другому администратору или диспетчеру. Каждый входит под своими учётными данными (никнейм + пароль). Несколько админов могут одновременно работать с приложением и следить за заказами и водителями — приложение не привязано к одному устройству.
