# Как загрузить Relaxe Drive на GitHub — по шагам

Сделать нужно **только 2 вещи**. Всё остальное уже настроено.

---

## Шаг 1. Создать репозиторий на сайте (1 раз)

1. Открой в браузере: **https://github.com/new**
2. В поле **Repository name** напиши: `Relaxe_Drive`
3. **Не ставь галочки**: не добавляй README, .gitignore, license (оставь всё выключенным).
4. Нажми зелёную кнопку **«Create repository»**.

Готово — репозиторий создан.

---

## Шаг 2. Отправить код с компьютера

1. Открой **Cursor** (или PowerShell) и открой папку проекта.
2. Открой терминал (Terminal → New Terminal).
3. Вставь эту команду и нажми Enter:

   ```
   git push -u origin main
   ```

4. Когда спросит **Username** — введи: `N1KTO-MHTM` и Enter.
5. Когда спросит **Password** — вставляй **не пароль от GitHub**, а **токен** (см. ниже, как создать).

После этого код появится на GitHub.

---

## Как создать токен (если спрашивает Password)

1. Зайди на: **https://github.com/settings/tokens**
2. Нажми **«Generate new token»** → **«Generate new token (classic)»**.
3. Название: например `Relaxe Drive`.
4. Поставь галочку **repo** (полный доступ к репозиториям).
5. Внизу нажми **«Generate token»**.
6. **Скопируй токен** (он показывается один раз). Вставляй его в терминал, когда просит Password.

---

Если на шаге 2 появится ошибка «Repository not found» — значит, шаг 1 ещё не сделан (репозиторий не создан). Сначала создай репозиторий на сайте, потом снова выполни `git push -u origin main`.
