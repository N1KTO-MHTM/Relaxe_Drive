# Screenshots — UI 1:1

Страница с разметкой и стилями дашборда **как в приложении** (шрифт Outfit, цвета, панели, кнопки).

## Как сделать скриншоты 1 в 1

1. Откройте в браузере файл:
   - **`dashboard-1to1.html`**

2. Переключение вида:
   - **Driver view** — `?view=driver` (или кнопка «Driver view»)
   - **Dispatcher view** — `?view=dispatcher` (или «Dispatcher view»)

3. Сделайте скриншот нужного вида (Win+Shift+S, или PrtScn, или инструменты разработчика → Capture screenshot).

Полный путь к файлу:
```
docs/screenshots/dashboard-1to1.html
```

Или откройте из корня проекта:
```
file:///.../Relaxe Drive/docs/screenshots/dashboard-1to1.html
```
